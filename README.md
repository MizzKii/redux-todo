# redux-todo
Todo list power by React and Redux.
# Requirement
  node (Recommend 6.6.0)
  npm  (Recommend 3.10.8)
# Installation
  <pre>
    npm install
  </pre>
# Start application
  <pre>
    npm run start
    or
    npm run dev
  </pre>
# Environment
  <pre>
    NODE_ENV = production // for production
    NODE_ENV = development // default for development
  </pre>
